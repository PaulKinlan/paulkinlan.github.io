--- layout: post title: A Minor Problem with my AJAX Application published: true --- It appears that my sample AJAX Application (here). Has some minor security issues. This is a similar problem to the problems I have in Firefox, cross site data access.

The work around for this is simple in IE6 and IE7 (although not preferable). If you go to "Internet Options -> Security -> Custom Level" and select the option under "Miscellaneous" select "Enable" in the "Access data sources across domains" section.
